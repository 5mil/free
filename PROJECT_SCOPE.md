# Project Scope: Motus Free Compute Resource Site

Build a public, general-purpose free computing resource site called Motus.
Organized as a markdown-based knowledge hub that can be rendered into a static website later.

## Core goals
- Curate free and low-cost compute resources across 25 categories.
- Organize content into numbered markdown files.
- Keep structure simple enough to maintain by hand.
- Support later expansion into a static site, dashboard, or searchable index.

## Naming and tone
- Name: Motus
- Tone: technical, organized, practical
- Format: concise markdown with links and short descriptions

## Next steps
- Populate each category file with curated resource lists and links.
- Add a homepage or index later if needed.
- Optionally add tags, search, and filters in a future version.
